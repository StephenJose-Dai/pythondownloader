# pythondownloader下载器

### 简介
一个基于Python写的单线程下载器，输入下载地址和下载次数就能下载，仅下载不保存任何下载的文件  
这主要是我用来测试下载速度用的。

### 运行方式

运行方式一共有2种

> 1、命令方式运行

>> ````

>> ````
